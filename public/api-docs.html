üîë Authentication Endpoints

POST /auth/register

Register a new user account and receive an API token

POST /auth/login

Login with existing credentials and receive an API token

GET /auth/user

Get authenticated user information

POST /auth/logout

Logout and revoke the current token

üîç SEO Analysis Endpoints

POST /v1/seo/analyze

Analyze a single URL for comprehensive SEO metrics

POST /v1/seo/analyze/batch

Analyze multiple URLs in batch (up to 10 URLs)

GET /v1/seo/history

Get analysis history for the authenticated user

GET /v1/seo/history/{id}

Get a specific analysis by ID

GET /v1/seo/status/{jobId}

Get status of an async analysis job

GET /v1/seo/batch/{batch_id}

Get batch analysis status by batch ID

üìù Analysis Options

When performing SEO analysis, you can include these optional parameters:

  • force_refresh: Force refresh of cached results
  • include_quality_analysis: Include content quality assessment
  • include_technical_analysis: Include technical SEO checks
  • include_content_analysis: Include content analysis
  • include_performance_analysis: Include performance metrics
  • javascript_enabled: Enable JavaScript when crawling
  • mobile_analysis: Perform mobile-specific analysis
  • timeout: Request timeout in seconds (5-60)

üîî Webhook Events

Webhooks can be configured to listen for these events:

  • analysis.completed: Single URL analysis completed successfully
  • analysis.failed: Single URL analysis failed
  • batch.completed: Batch analysis completed successfully
  • batch.failed: Batch analysis failed

Webhook payloads include event metadata and analysis results. Optionally configure a secret for payload verification using HMAC-SHA256.

üöÄ Getting Started

  1. Import the Postman Collection: Download and import the collection above into Postman
  2. Set Environment Variables: Configure base_url (e.g., http://localhost:8000/api)
  3. Register/Login: Use the authentication endpoints to get a Bearer token
  4. Set Auth Token: The token will be automatically saved to the auth_token variable
  5. Start Analyzing: Use the SEO analysis endpoints to analyze your URLs
