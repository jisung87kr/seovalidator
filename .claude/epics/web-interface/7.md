# Task: API 기본 구조 및 인증 시스템

## Technical Details

### API Structure
```
/api/v1/
├── auth/
│   ├── register          # API 키 발급
│   ├── revoke           # API 키 삭제
│   └── refresh          # API 키 갱신
├── seo/
│   ├── analyze          # SEO 분석 요청
│   ├── result/{id}      # 분석 결과 조회
│   └── history          # 분석 히스토리
├── system/
│   ├── health           # 헬스체크
│   └── status           # 시스템 상태
└── docs/                # API 문서
```

### Authentication Flow
```php
// API Key 기반 인증
X-API-Key: your-api-key-here

// Rate Limiting
X-RateLimit-Limit: 100
X-RateLimit-Remaining: 99
X-RateLimit-Reset: 1632150000
```

### Response Format
```json
{
  "success": true,
  "data": {
    // 응답 데이터
  },
  "message": "Success",
  "meta": {
    "version": "1.0",
    "timestamp": "2025-09-18T10:15:00Z",
    "request_id": "uuid"
  }
}
```

### Error Response Format
```json
{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "The given data was invalid.",
    "details": {
      "url": ["The url field is required."]
    }
  },
  "meta": {
    "version": "1.0",
    "timestamp": "2025-09-18T10:15:00Z",
    "request_id": "uuid"
  }
}
```

## Implementation Plan

### 1. Laravel Sanctum 설정
```bash
composer require laravel/sanctum
php artisan vendor:publish --provider="Laravel\Sanctum\SanctumServiceProvider"
php artisan migrate
```

### 2. API 키 모델 생성
```php
// app/Models/ApiKey.php
class ApiKey extends Model
{
    protected $fillable = [
        'name', 'key', 'user_id', 'abilities', 
        'rate_limit', 'expires_at', 'last_used_at'
    ];
    
    protected $casts = [
        'abilities' => 'array',
        'expires_at' => 'datetime',
        'last_used_at' => 'datetime'
    ];
}
```

### 3. Rate Limiting 설정
```php
// app/Http/Kernel.php
'api' => [
    'throttle:api',
    \Laravel\Sanctum\Http\Middleware\EnsureFrontendRequestsAreStateful::class,
    'bindings',
],

// config/rate-limiting.php
'api' => [
    'free' => '100:hour',
    'premium' => '1000:hour',
    'burst' => '10:minute'
]
```

### 4. 미들웨어 구현
- `ApiKeyAuthentication`: API 키 검증
- `ApiRateLimit`: 계층별 속도 제한
- `ApiLogging`: API 사용 로깅
- `ApiResponse`: 응답 형식 표준화

### 5. 컨트롤러 구조
```php
// app/Http/Controllers/Api/V1/
├── AuthController.php
├── SeoController.php
├── SystemController.php
└── BaseApiController.php
```
