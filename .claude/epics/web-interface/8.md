### Queue Job Implementation

```php
// app/Jobs/AnalyzeSeoJob.php
class AnalyzeSeoJob implements ShouldQueue
{
    use Dispatchable, InteractsWithQueue, Queueable, SerializesModels;
    
    public function __construct(
        public string $analysisId,
        public string $url,
        public array $keywords = [],
        public array $options = []
    ) {}
    
    public function handle(SeoAnalyzer $analyzer): void
    {
        $analysis = SeoAnalysis::findOrFail($this->analysisId);
        
        try {
            // 진행률 업데이트
            $this->updateProgress($analysis, 0, 'Starting analysis...');
            
            // SEO 분석 실행
            $result = $analyzer->analyze($this->url, $this->keywords, $this->options);
            
            // 결과 저장
            $analysis->update([
                'status' => 'completed',
                'results' => $result,
                'completed_at' => now()
            ]);
            
        } catch (\Exception $e) {
            $analysis->update([
                'status' => 'failed',
                'error_message' => $e->getMessage(),
                'completed_at' => now()
            ]);
            
            throw $e;
        }
    }
}
```

### Request Validation

```php
// app/Http/Requests/Api/AnalyzeSeoRequest.php
class AnalyzeSeoRequest extends FormRequest
{
    public function rules(): array
    {
        return [
            'url' => [
                'required',
                'url',
                'max:2048',
                'regex:/^https?:\/\//'
            ],
            'keywords' => 'sometimes|array|max:10',
            'keywords.*' => 'string|max:100',
            'options.include_technical' => 'sometimes|boolean',
            'options.include_content' => 'sometimes|boolean',
            'options.language' => 'sometimes|in:auto,en,ko'
        ];
    }
    
    public function messages(): array
    {
        return [
            'url.required' => 'URL is required for SEO analysis',
            'url.url' => 'Please provide a valid URL',
            'url.regex' => 'URL must start with http:// or https://',
            'keywords.max' => 'Maximum 10 keywords allowed',
            'keywords.*.max' => 'Each keyword must be less than 100 characters'
        ];
    }
}
```

### API Resource Classes

```php
// app/Http/Resources/Api/SeoAnalysisResource.php
class SeoAnalysisResource extends JsonResource
{
    public function toArray($request): array
    {
        return [
            'analysis_id' => $this->id,
            'url' => $this->url,
            'status' => $this->status,
            'overall_score' => $this->score,
            'grade' => $this->score ? $this->getGrade() : null,
            'created_at' => $this->created_at->toISOString(),
            'completed_at' => $this->completed_at?->toISOString(),
            'results' => $this->when($this->isCompleted(), $this->results),
            'error_message' => $this->when($this->isFailed(), $this->error_message)
        ];
    }
}
```

## Error Handling

### Error Codes
- `INVALID_URL`: 유효하지 않은 URL
- `URL_UNREACHABLE`: 접근 불가능한 URL
- `ANALYSIS_FAILED`: 분석 실행 오류
- `ANALYSIS_NOT_FOUND`: 분석 결과 미존재
- `QUOTA_EXCEEDED`: API 할당량 초과
- `INVALID_PARAMETERS`: 잘못된 매개변수
