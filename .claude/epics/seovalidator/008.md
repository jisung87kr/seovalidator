## Technical Details
### Implementation Approach
- Laravel API Resources for response formatting
- Sanctum for API authentication
- Rate limiting middleware
- OpenAPI 3.0 specification
- Webhook system for real-time notifications

### API Endpoints
```php
// Authentication
POST /api/auth/login
POST /api/auth/refresh
POST /api/auth/logout

// Analysis
POST /api/v1/analyze
POST /api/v1/analyze/batch
GET /api/v1/analysis/{id}
GET /api/v1/analysis/{id}/status
DELETE /api/v1/analysis/{id}

// Reports
GET /api/v1/analysis/{id}/report
GET /api/v1/analysis/{id}/report/pdf
GET /api/v1/analysis/{id}/report/csv

// Account
GET /api/v1/account/usage
GET /api/v1/account/limits
GET /api/v1/webhooks
POST /api/v1/webhooks
```

### API Response Format
```json
{
  "success": true,
  "data": {...},
  "meta": {
    "version": "1.0",
    "timestamp": "2025-09-17T13:35:09Z",
    "request_id": "uuid"
  },
  "errors": null
}
```

### Authentication & Security
- API 키 헤더: `Authorization: Bearer {api_key}`
- Rate limiting: 1000 requests/hour (기본)
- IP 화이트리스트 지원
- Request/Response 로깅

### Code Locations
- `routes/api.php` - API 라우팅
- `app/Http/Controllers/Api/` - API 컨트롤러들
- `app/Http/Resources/` - API 리소스 클래스들
- `app/Http/Middleware/` - API 미들웨어
- `docs/api/` - API 문서
