# Task: Content SEO Analyzer

## Description
웹페이지의 콘텐츠 SEO 요소를 분석하는 서비스를 구현합니다. 키워드 밀도, 가독성, 콘텐츠 구조, 내부/외부 링크를 분석합니다.

## Technical Details
### Implementation Approach
- 자연어 처리 기법을 활용한 콘텐츠 분석
- 한국어/영어 텍스트 처리 최적화
- 가독성 지수 계산 (Flesch Reading Ease 등)
- 키워드 밀도 및 TF-IDF 분석

### Key Features
```php
// ContentSeoAnalyzer
- analyzeKeywordDensity(string $content, array $keywords): KeywordResult
- analyzeReadability(string $content): ReadabilityResult
- analyzeHeadingStructure(CrawlResult $result): HeadingResult
- analyzeImageOptimization(CrawlResult $result): ImageResult
- analyzeLinks(CrawlResult $result): LinkResult
- analyzeContentQuality(string $content): ContentQualityResult
- calculateContentScore(array $results): int

// Analysis Results
- KeywordResult: density, distribution, primary_keywords, suggestions
- ReadabilityResult: score, sentence_length, word_count, complexity
- HeadingResult: h1_count, hierarchy_issues, heading_distribution
- LinkResult: internal_count, external_count, broken_links, anchor_text
```
