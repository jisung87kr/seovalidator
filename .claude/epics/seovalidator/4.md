# Task: Web Crawler Service Foundation

## Description

## Technical Details
### Implementation Approach
- spatie/browsershot 패키지 활용
- Chrome/Chromium 헤드리스 브라우저 사용
- Service Container를 통한 의존성 주입
- Interface 기반 설계로 테스트 용이성 확보

### Key Components
```php
// UrlCrawlerService
- crawl(string $url): CrawlResult
- extractMetaTags(DOMDocument $dom): array
- extractImages(DOMDocument $dom): array
- extractLinks(DOMDocument $dom): array
- validateUrl(string $url): bool

// CrawlResult DTO
- url, title, meta_description, html_content
- images, internal_links, external_links
- response_time, status_code, error_message
```

### Code Locations
- `app/Services/UrlCrawlerService.php`
- `app/Contracts/CrawlerInterface.php`
- `app/DTOs/CrawlResult.php`
- `config/crawler.php` - 설정 파일
