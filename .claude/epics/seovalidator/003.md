# Task: Technical SEO Analyzer

## Description
웹페이지의 기술적 SEO 요소를 분석하는 서비스를 구현합니다. 메타태그, 구조화 데이터, 페이지 속도, 모바일 친화성 등을 검증합니다.

## Acceptance Criteria
- [ ] TechnicalSeoAnalyzer 서비스 구현
- [ ] 메타태그 분석 (title, description, keywords, Open Graph)
- [ ] 구조화 데이터 검증 (JSON-LD, Microdata)
- [ ] 페이지 속도 측정 및 개선점 제시
- [ ] 모바일 친화성 검사
- [ ] SSL 인증서 및 보안 헤더 확인
- [ ] H1-H6 태그 구조 분석
- [ ] 이미지 alt 텍스트 검증
- [ ] 분석 결과 점수화 알고리즘

## Technical Details
### Implementation Approach
- Strategy Pattern으로 각 분석 유형별 독립적 구현
- Google PageSpeed Insights API 연동
- DOM 파싱을 통한 메타데이터 추출
- 가중치 기반 점수 계산

### Key Features
```php
// TechnicalSeoAnalyzer
- analyzeMetaTags(CrawlResult $result): MetaTagsResult
- analyzeStructuredData(CrawlResult $result): StructuredDataResult
- analyzePageSpeed(string $url): PageSpeedResult
- analyzeMobileFriendly(CrawlResult $result): MobileFriendlyResult
- analyzeSecurityHeaders(CrawlResult $result): SecurityResult
- calculateTechnicalScore(array $results): int

// Analysis Results
- MetaTagsResult: title_length, description_length, og_tags, twitter_cards
- PageSpeedResult: score, load_time, core_vitals, recommendations
- SecurityResult: ssl_status, security_headers, https_redirect
```
