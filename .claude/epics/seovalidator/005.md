# Task: SEO Scoring System

## Description
기술적 SEO와 콘텐츠 SEO 분석 결과를 종합하여 0-100점 범위의 전체 SEO 점수를 계산하는 시스템을 구현합니다. 가중치 기반 점수 계산과 개선 제안을 제공합니다.

## Technical Details
### Implementation Approach
- 가중치 기반 점수 계산 (configurable weights)
- Rule Engine 패턴으로 개선 제안 생성
- 점수 히스토리 추적 및 트렌드 분석
- A/B 테스트를 위한 다중 점수 알고리즘 지원

### Key Components
```php
// SeoScoreCalculator
- calculateOverallScore(TechnicalResult $tech, ContentResult $content): SeoScore
- calculateTechnicalScore(TechnicalResult $result): int
- calculateContentScore(ContentResult $result): int
- generateSuggestions(SeoScore $score): array
- getScoreGrade(int $score): string
- compareScores(SeoScore $current, SeoScore $previous): ScoreComparison

// Scoring Configuration
- technical_weight: 0.6 (60%)
- content_weight: 0.4 (40%)
- category_weights: meta_tags(20%), page_speed(25%), content_quality(30%)
```

### Scoring Algorithm
```
총점 = (기술적 SEO × 0.6) + (콘텐츠 SEO × 0.4)

기술적 SEO:
- 메타태그 품질: 20점
- 페이지 속도: 25점
- 모바일 친화성: 15점
- 보안/구조: 15점
- 구조화 데이터: 10점

콘텐츠 SEO:
- 키워드 최적화: 15점
- 가독성: 10점
- 콘텐츠 품질: 10점
- 링크 품질: 5점
```
