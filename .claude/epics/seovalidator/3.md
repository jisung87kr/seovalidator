# Task: Database Models and Migrations

## Description
Laravel 기반 SEO Validator의 핵심 데이터 모델과 데이터베이스 스키마를 설계하고 구현합니다. SEO 분석 결과를 저장하기 위한 테이블 구조와 Eloquent 모델을 생성합니다.

## Acceptance Criteria
- [ ] SeoAnalysis 모델 및 마이그레이션 생성
- [ ] UrlAnalysis 모델 및 마이그레이션 생성
- [ ] TechnicalSeoResult 모델 및 마이그레이션 생성
- [ ] ContentSeoResult 모델 및 마이그레이션 생성
- [ ] ApiKey 모델 및 마이그레이션 생성
- [ ] 모든 테이블 관계 정의 (foreign keys, indexes)
- [ ] 데이터베이스 시딩 파일 작성
- [ ] 모델 간 관계 메서드 구현

## Technical Details
### Implementation Approach
- Laravel Eloquent ORM 사용
- Repository Pattern 적용을 위한 기반 구조
- JSON 컬럼 활용으로 유연한 데이터 저장
- 적절한 인덱싱으로 성능 최적화

### Key Models Structure
```php
// SeoAnalysis: 분석 세션 관리
- id, url, status, score, started_at, completed_at
- hasMany: UrlAnalysis, TechnicalSeoResult, ContentSeoResult

// UrlAnalysis: URL별 분석 결과
- id, seo_analysis_id, url, title, meta_description, status_code

// TechnicalSeoResult: 기술적 SEO 검증 결과
- id, seo_analysis_id, meta_tags (JSON), page_speed, mobile_friendly

// ContentSeoResult: 콘텐츠 SEO 분석 결과
- id, seo_analysis_id, keyword_density (JSON), readability_score, h_tags (JSON)

// ApiKey: API 인증 관리
- id, key, name, rate_limit, expires_at, is_active
```

### Code Locations
- `database/migrations/` - 마이그레이션 파일들
- `app/Models/` - Eloquent 모델들
- `database/seeders/` - 시딩 파일
- `database/factories/` - 모델 팩토리
