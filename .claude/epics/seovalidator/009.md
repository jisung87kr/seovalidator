# Task: Report Generation System

## Description
SEO 분석 결과를 PDF, Excel, CSV 형태로 내보낼 수 있는 리포트 생성 시스템을 구현합니다. 템플릿 기반의 전문적인 리포트를 제공합니다.

## Acceptance Criteria
- [ ] ReportGenerator 서비스 구현
- [ ] PDF 리포트 생성 (DomPDF 활용)
- [ ] Excel 리포트 생성 (PhpSpreadsheet 활용)
- [ ] CSV 리포트 생성
- [ ] 리포트 템플릿 시스템
- [ ] 차트 및 그래프 포함 (이미지 생성)
- [ ] 브랜딩 및 로고 삽입
- [ ] 대량 리포트 생성 (배치 처리)
- [ ] 리포트 다운로드 링크 및 만료 처리

## Technical Details
### Implementation Approach
- Template-based report generation
- Queue system for heavy report processing
- Chart generation using server-side libraries
- File storage with expiration management
- Multi-format export support

### Report Types
```php
// PDF Report
- Executive Summary (1페이지)
- Technical SEO Analysis (2-3페이지)
- Content SEO Analysis (2-3페이지)
- Recommendations (1-2페이지)
- Appendix (상세 데이터)

// Excel Report
- Summary Sheet
- Technical Analysis Sheet
- Content Analysis Sheet
- Raw Data Sheet
- Charts and Graphs

// CSV Report
- Flat structure for data analysis
- Multiple CSV files (분석 영역별)
```

### Key Components
```php
// ReportGenerator
- generatePdf(SeoAnalysis $analysis): string
- generateExcel(SeoAnalysis $analysis): string
- generateCsv(SeoAnalysis $analysis): array
- generateCharts(SeoAnalysis $analysis): array

// Template Engine
- PdfTemplate: PDF 레이아웃 및 스타일
- ExcelTemplate: Excel 워크시트 구조
- ChartGenerator: 차트 이미지 생성

// Storage Management
- storeReport(string $content, string $format): string
- getDownloadUrl(string $filename): string
- cleanupExpiredReports(): void
```

### Report Features
- 회사 로고 및 브랜딩
- SEO 점수 시각화 (차트, 게이지)
- 개선 제안 우선순위 표시
- 경쟁사 비교 (선택사항)
- 히스토리 트렌드 분석

### Code Locations
- `app/Services/ReportGenerator.php`
- `app/Templates/` - 리포트 템플릿들
- `resources/views/reports/` - PDF 템플릿 뷰
- `storage/app/reports/` - 생성된 리포트 저장
- `app/Jobs/GenerateReportJob.php`
